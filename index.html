  • Projects

    Chord Extractor

    I am currently working on this project under a competition India Innovation Challenge Design Contest '18, jointly organised by Texas Instruments and the Govt. of India. In this project I used machine learning based models to extract chords out of a music file. To complete this task I extracted 12 dimensional Pitch Class Profile vectors from audio file using Fourier Transformation. I achieved 95% training accuracy and 86% test accuracy on a dataset of 2000 chords. Here is the github link to my project.

    Heart Rate Monitor

    I completed this project during Spring Semester 2017 under Prof. Amitabh Sanyal. I implemented a chat bot that uses Hill Encryption to encrypt and decrypt messages on the user end only and deployed various methods to keep the key visible only to the interacting users (not even to the system). I also made a chatbox so that users can keep track of their chats while interacting with other users. Here is the github link.

    Movie Reviewer

    I did this self project during summers and got it evaluated later by Web and Coding Club, IIT bombay. In this project I deployed a rating system based on comments of user on the movie. I used TextBlob library for this purpose and extracted Subjectivity and Polarity to use them as features to train them on neural network. I further regulated these values to obtain final star rating of the movie. Here is the github link.
